[Axiora](https://axiora.dev) provides structured financial data for Japanese listed companies. Data comes from EDINET (金融庁) filings — XBRL-parsed, normalized across three accounting standards (J-GAAP, IFRS, US-GAAP), and served as typed JSON.

## Installation

<CodeGroup>
```bash pip
pip install -U langchain-axiora
```
```bash uv
uv add langchain-axiora
```
</CodeGroup>

## Setup

Sign up at [axiora.dev](https://axiora.dev) for a free API key (no credit card required).

Set the key as an environment variable:

```python
import os
os.environ["AXIORA_API_KEY"] = "ax_live_..."
```

Or pass it directly when initializing:

```python
from langchain_axiora import AxioraToolkit

toolkit = AxioraToolkit(api_key="ax_live_...")
```
